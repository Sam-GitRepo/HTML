Lets start with the basics of HTML

Meta tags

Meta Tag is Generally used for SEO(Search engine optimization).
There are various types of meta tags

  • charset="UTF-8"

    • It simply means that character declare the page character encoding

  • name="viewport" content="width=device-width, initial-scale=1.0

    • Making website responsive & adjust width thats looks good in phone or PC

Headings Tags

There are basically 6 types of heading tag H1 to H6
H1 is the largest & H6 is is the smallest

Paragraphs tags

Paragraph tag is use to write new Paragraphs for jump on new line. if need 4 different lines of Paragraphs tags simply write p*4.

Some other tags using in Paragraphs

Strong Tag - Writing some text as bold.
em Tag - This is use for convert into italic font.
br Tag - It is a self closing tag to change a line in a paragraph.
hr Tag - It is also a self closing tag to add a horizontal ruler in webpage.
