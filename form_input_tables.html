Tables

Tables is a just a combination of rows & columns on a web page. The main part is the table tag and it consist of two part the table head and table body

table - for define table

th -for define header cell

tr -for define row cell

td - for define cell in a table

Attributes of use inside th tag

abbr - it is used for abbreviated version of text content

colspan - It is used for defining numbers of columns a header cell

header - It is used to specify one or more header cells

rowspan - It is used to set number of rows a header cell

Scope - It is used to specify the scope of header content
